Installation

The most recent installation instructions can always be found in the main README.rdoc file distributed with Blacklight. You can read this file online at http://github.com/projectblacklight/blacklight (Scroll down to the bottom of the page).

One can also install a virtual appliance of Blacklight. The appliance is located at http://www.projectblacklight.org/downloads/blacklight.zip. It is designed for VirtualBox, but with a few modifications in can run on VMWare. Simply import the appliance in to VirtualBox and start it. The default password is blacklight. Once it is started point, you can access Blacklight by pointing your browser to the IP address of the appliance, which can be found by the command ifconfig.
